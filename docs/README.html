# VIPTV - 工作室

工作室介绍： 本工作室由多个资深站长自由组建的,主要以ThinkPHP为基础开发后端,VueJs框架为基础开发前端,为用户提供优质程序代码.多年来,我们一直坚持完善和迭代更新，在业内拥有领先的市场占有率和良好的用户口碑，相信是你值得信赖

# 📥 如何本地开发

# 克隆本仓库
$ git clone https://gitee.com/viptv/viptv.git

# 安装依赖
$ npm install
# 或者使用 yarn
$ yarn install

# 启动开发服务器
$ yarn dev
